In VS create Razor pages app and add container support (Dockerfile)
Build and run on Docker desktop (for release not debug, otherwise dev tag on image)


docker tag hellorazorcontainerapp:latest gclynch.azurecr.io/gclynch/hellorazorcontainerapp

az login
az acr login -n gclynch
docker push gclynch.azurecr.io/gclynch/hellorazorcontainerapp
az acr repository list --name gclynch --output table



in portal create the container app:
pick gclynch.acr.io
pick image  hellorazorcontainerapp
auth type: managed identity
ingress - traffic from anywhere

or do it in VS
- can publish image to container registry
- can publish app to container app
